# Device Overview

The overview page displays the following information:

* **Device name**
* **Screenshot**
* **Manufacturer**
* **Model name**
* **Connected Wi-Fi network name**
* **Wi-Fi connection IP address**
* **Android system version**
* **Battery level**
* **Internal storage usage**
* **Memory usage**

## Change device name

By default, the device name uses the device's Bluetooth name.

To change it, you can either:

import FluentEdit20Regular from '~icons/fluent/edit-20-regular';

* Change the device name in device Bluetooth setting page.
* Tap the <FluentEdit20Regular/> button, and set a nickname. This name is only used in Tango.

## Storage and memory usage

These values can be slightly different from the values displayed in the device's "Settings" app. Some storage and memory is reserved for the system, and not counted in the values displayed here.
